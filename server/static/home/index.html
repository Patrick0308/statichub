FAQ

Do I need to use AI to use Statichub?

No. AI-assisted and pure CLI workflows are both first-class.

Can I deploy plain HTML, JS, and CSS?

Yes. Any static output folder can be deployed.

Will this replace my existing subdomain sites?

No. Base-domain homepage routing is separate from project subdomain serving.
