Install

Pick your OS command.

Recommended command

brew install statichub/tap/statichub

Fallback: download tarball and place `statichub` in your PATH.

FAQ

Should I use Skill or CLI mode?

Use Skill for AI pair workflows and CLI for direct terminal-first control.

What if install commands fail?

Use each OS panel fallback method, then verify with statichub --version.

How do upgrades work?

Re-run your install method or pull the latest release from the release notes page.

How do I fix auth issues?

Run statichub auth status and sign in again if your session expired.
